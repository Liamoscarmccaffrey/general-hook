GENERAL HOOK

        
Build and test APIs instantly — no account, no backend, no setup.

        

          General Hook is a free mock API server and webhook inspector that runs entirely in your browser.
          Define custom endpoints with JSON responses, import an OpenAPI or Postman spec, or spin up a
          public webhook receiver to inspect live payloads — all without installing anything or creating an account.
        

        

          Powered by WebAssembly, your server runs directly inside this browser tab.
          Nothing is sent to an external server and no data leaves your device.
        

        

          Your server only lives as long as this tab is open.
          Closing or refreshing this page will immediately shut down any running API or webhook.
          To start a fresh server with a new configuration, simply refresh the tab.
